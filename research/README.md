# 🧪 Travaux de Recherche et Expérimentation
## Dossier de Transition et Archives

Ce dossier contient tout le travail exploratoire, les scripts de phase originaux et les notebooks ayant servi à la mise au point du pipeline **SON 4G/LTE**.

### 📂 Structure du dossier
*   **`notebooks/`** : Exploration de données (EDA), prototypage des modèles et rapports de benchmark originaux.
*   **`legacy_scripts/`** : Scripts `phase1` à `phase5` originaux. Ces scripts sont conservés pour archive mais la production utilise désormais le package `src/`.
*   **`debug/`** : Scripts utilitaires pour le débogage des matrices radio et des étapes de simulation.
*   **`verification/`** : Contient `critical_verification.py`, l'outil de validation de la conservation de masse.
*   **`data/`, `models/`, `config/`, `offline/`, `reports/`** : Assets et artefacts générés lors des phases de recherche.
