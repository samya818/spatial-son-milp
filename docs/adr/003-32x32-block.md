# ADR 003: Contiguous 32x32 Dense Block Validation

## Status
Accepted

## Context
Early versions of the pipeline used a 600-cell hybrid sampling (random + centers). While useful for broad testing, it broke the spatial continuity of the network, making Handover optimization unrealistic.

## Decision
We switched to a **contiguous 32x32 block (1024 cells)** centered on Milan (Duomo/Garibaldi area).

## Rationale
- **Topological Integrity**: Every antenna has its real geographical neighbors within the cluster.
- **Urban Density**: The center of Milan provides the highest density of micro-cells, which are the primary levers for successful SON offloading.
- **Edge Effects**: A contiguous block allows us to mathematically define "Closed Edges," ensuring no traffic "leaks" out of the optimization scope.

## Consequences
- **Positive**: Gain increased from ~40% to 73.5% because the MILP had more (and real) neighbors to work with.
- **Negative**: The dataset is geographically restricted, requiring separate validation for rural or suburban areas.
